# 投資インテリジェンス ダッシュボード (FIN-INTELLIGENCE)

## 概要
個人投資家向けのAI駆動型金融情報ダッシュボードです。指定された銘柄の最新状況や、市場に影響を与える重要ニュースをGemini AIが自動で収集・選定し、分かりやすい形式で提供します。ゴシップやノイズとなる情報を排除し、投資判断に必要な情報（業績・業務提携・マクロ経済など）だけを素早くキャッチできるように設計されています。

## 主な機能
- **🤖 AIニュース選定・要約モジュール**: 膨大なニュースから重要な金融・経済・企業動向に関するニュースを厳選し、客観的な事実のみを「3箇条書き」で要約して表示します。「決算・業績」や「マクロ経済」などのカテゴリごとに色分けして表示します。
- **📈 ウォッチリスト管理 (AI自動取得)**: 銘柄名（例: トヨタ自動車）を入力するだけで、AIが自動的に直近の株価と前日比を推定・取得し、リストに追加します。
- **🔄 ダッシュボード更新**: ダッシュボードのタイトル横にある「更新」ボタンから、現在のウォッチリストの状態に合わせて、最新の市場概況（日経平均・ドル円）、ニュース、およびウォッチリストの株価情報を一括で再取得できます。
- **🎌 市場概況の表示**: 日経平均株価やUSD/JPY為替レートなどのマクロ指標をヘッダーに常時表示します。

## 技術スタック
- **フロントエンド**: React 19, TypeScript, Vite
- **スタイリング**: Tailwind CSS v4
- **アイコン**: Lucide React
- **AI / API**: Google Gen AI SDK (`@google/genai`), Gemini API (Gemini 1.5 / 3.0 系モデルを利用し、データ取得・ニュース要約・JSON構造化バックエンドとして活用)

## 使い方
1. **ダッシュボードの閲覧**: アプリを開くと、最新の市場概況と重要ニュース3件が自動的に生成され、カード形式で表示されます（ロード中はスピナーが表示されます）。
2. **ウォッチリストへの銘柄追加**: 画面左側のサイドバー（ウォッチリスト）にある「AI自動取得ツール」のテキストボックスに、気になる企業名（例: 「任天堂」など）を入力し、「追加」ボタンを押します。AIが株価情報を自動的に取得してリストに追加します。
3. **ウォッチリストからの削除**: リスト上の各銘柄にマウスカーソル（ホバー）を合わせると表示される「×」ボタンをクリックすると、対象の銘柄をリストから除外できます。
4. **最新情報の手動更新**: ニュースエリア上部の「更新」ボタンをクリックすると、最新のニュースと市場概況、登録されているスマートウォッチリストの状況を一括でAIに再リクエストし画面を更新します。
